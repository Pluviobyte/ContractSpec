# Changelog

## 0.2.0

- Added `contracts/bindings.md` as the executable contract bridge.
- Upgraded the `contract-spec` schema to version 2.
- Added a bindings template and create-task binding example.
- Added `contractspec validate --change`, `--strict`, `check-bindings`, and `doctor`.
- Updated the workflow around Contract Gate, Binding Gate, Failing Test Gate, Implementation Gate, and Verification Gate.
- Added test-first guidance: tests are executable contracts.

## 0.1.0

- Initial OpenSpec-compatible ContractSpec schema bundle.
- Added full-stack contract templates, create-task example, validator, prompt, and CI.
